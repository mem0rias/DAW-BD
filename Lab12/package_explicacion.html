Que es package.json

  • Name: contiene el nombre que se le da al paquete de npm
  • Version: Contiene la version de tu paquete, para versionar y documentar cambios
  • Descripcion: String que contiene palabras clave de tu proyecto. se usa para encontrar el paquete con npm search
  • Keywords: Mismo funcionamiento que descripcion, sirve para que el usuario encuentre el paquete y para que npm search lo encuentre
  • homepage: El URL que contiene la liga del proyecto como un enlace al repositorio de git del proyecto
  • bugs: Contiene ligas de contacto para reportar bugs del proyecto
  • License: Apartado que contiene la licencia del paquete, permite al usuario saber como puede usar el software
  • Author: Contiene la informacion del autor y colaboradores del proyecto
  • Funding: Tiene ligas para apoyar el desarrollo del proyecto
  • main: Establece el archivo que sera el punto de entrada del sistema.

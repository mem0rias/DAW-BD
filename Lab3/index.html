Preguntas a responder

  1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    1. Al usar important, para sobreescribir atributos de CSS? al forzar su aplciacon con la palabra clave "!important" esto es util para forzar estilos de etiquetas en clases especificas

  2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    1. Se debe de escoger cuidadosamente el tamaño de la imagen, ya que si es mas pequeña que la pagina la imagen se comenzara a repetir hasta que se llene la pantalla. Para evitar esta situacion se debe utilizar "background-repeat: no repeat".

  3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    1. Yo recomedaria utilizar %, ya que puede escalar de una mejor manera en dispositivos con pantallas de diferente tamaño

  4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    1. El rendimiento es mejor ya que al utilizar una version minimizada se reduce la cantidad de informacion que se tiene que descargar de un servidor.

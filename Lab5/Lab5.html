Preguntas de Laboratorio

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Tienen en comun el nombre
    • Ambos lenguajes siguen el paradigma de POO
    • El codigo de java se compila mientras que el de JS es interpretado
    • JS los tipos de datos son dinamicos, es decir no se tiene que especificar, a diferencia de java, donde siempre se tiene que definir un tipo de dato para cada variable.
    • Java se utiliza para realizar aplicaciones tanto en locales, como web. Mientras que JavaScript funciona principalmente en conjunto con HTML y navegadores web
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • getTime() para obtener el tiempo transcurrido desde 01/01/1970
    • toDateString() para obtener la fecha actual en un formato legible
    • getMonth() Obtiene el mes dentro de un numero de 0 a 11
    • getHours() Obtiene la hora dentro de un rango de 0 a 23 Hrs
    • getMinutes() Obtiene los minutos dentro de un rango de 0 a 59 mins
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    • at(), retorna el elemento en la posicion especificada
    • concat(), retorna un arreglo nuevo con los elementos especificdos añadidos, ya sea otro arreglo o numeros extra
    • fill(), rellena un arreglo con un numero definido
    • find(), encuentra el primer elemento que coincida con la condicion dada
    • findLast(), encuentra el ultimo elemento que satisfaga la condicion dada
  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    • Se declara escribiendola dentro de una funcion.
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    • La variable global puede ser editada por cualquier parte del codigo por lo que hay que tener cuidado con que se modifique accidentalmente en alguna otra parte del codigo para evita problemas.
  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    • El metodo .search()
    • Este metodo es util ya que se pueden obetener informacion especifica de un string, como nombres direcciones de correo, combinaciones de palabras indecentes o prohibidas por la aplicacion web, etc.
